## 1. Sobre o Projeto
Este repositório contém os artefatos exigidos para a avaliação final da disciplina de Reuso de Software. O objetivo do trabalho foi selecionar um sistema legado real (Open Source), realizar um diagnóstico de reusabilidade baseado em métricas de engenharia de software (Complexidade Ciclomática, Acoplamento, Coesão) e aplicar refatorações para modernizar a arquitetura e promover o reuso.
